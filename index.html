Projects

Project 1

BookTown

The objective of this project was to create a book-reading website for all readers to easily access and read books from anywhere anytime. Used HTML, CSS, JavaScript, Bootstrap, Local server XAMPP, Firebase Storage for storing books, and Local Storage for user signup and login Through this project, we were able to successfully create a website where e-books come with an extensive range of multimedia tools which further increase the attractiveness of such books amongst young readers. Therefore, the website BookTown will provide readers with different genres of eBooks in just one click and caters to all types of readers - whether they are voracious readers looking for their next epic mystery or someone wanting to unwind with a light-hearted romance.

Project 2

Hospital Management System

This project aimed at designing a system to assist healthcare professionals in efficiently managing patient data, administrative tasks, doctor schedules, and more. The system was built using Python and integrated with MS Excel for data storage and analysis, facilitating streamlined operations within hospitals and clinics. Through this project, we were able to successfully create a system that can be a powerful tool to help healthcare centers improve their processes and streamline their operations. It is capable of providing an integrated, comprehensive solution to managing patient records, billing, and scheduling, as well as providing real-time insights into hospital performance. The system has the facility to give a unique ID for every patient and store the details of every patient and staff.

Project 3

Library Management System

The aim of this project was to design a Library Management System and help the admin team track the library books and documents. Used Python programming language to develop this project. Through this project, I was able to successfully develop a system that allows the admin to store the book details and the person's details and also allows storing details of all the data related to the library. The implementation of this system reduced manual data entry and provided readily calculated reports.
